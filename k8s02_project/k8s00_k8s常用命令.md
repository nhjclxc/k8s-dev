# k8s常用命令

## 1️⃣ Pod（最小运行单元）

运行容器的地方（真正干活的），一个 Pod = 一个或多个容器

| 命令                                                             | 解释                                             |
|----------------------------------------------------------------|------------------------------------------------| 
| kubectl get pods                                               | 查看当前上下文默认命名空间的所有pod                            | 
| kubectl get pods -n <text-namespace>                           | 查看指定命名空间的所有pod                                 | 
| kubectl get pods --all-namespaces                              | 查看所有命名空间的所有pod                                 | 
| kubectl get pods -A                                            | 查看所有命名空间的所有pod                                 | 
| kubectl config current-context                                 | 查看当前 context 的 namespace                       | 
| kubectl config set-context --current --namespace=<text-namespace> | 设置当前 context 的默认 namespace                     | 
| kubectl describe pod <pod-name>                                | 查看这个 Pod 的“完整详细状态 + 事件 + 调度 + 容器信息 + 错误原因”     | 
| kubectl logs <pod-name>                                   | 查看这个 Pod 的日志，查看实时日志：kubectl logs -f <pod-name> | 
| kubectl exec -it <pod-name> -- sh                                   | 进入正在运行的 Pod 容器内部，并打开一个 shell 交互终端              | 
| kubectl exec -it <pod> -c <container-name> -- sh                                   | 进入正在运行的 Pod 的其中一个容器内部，并打开一个 shell 交互终端         | 
| kubectl get pod <pod-name> -o yaml                                   | 查看置顶 Pod 的 YAML（完整配置）                          | 
|                                    |                                                |
